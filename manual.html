XO to SEQTRAK Manual

Export samples and MIDI from XO, convert the MIDI, and import the result into YAMAHA SEQTRAK.

Converter SMF Debugger

1. Prepare XO

  1. Open the XO plug-in or XO standalone application.
  2. Open Audio & MIDI Setup and set Sample Rate to 44100 Hz.
    Important: If this setting is different, SEQTRAK may not be able to load the exported samples.
    • The available sample rate options depend on the audio interface you use.
    • If you use SEQTRAK (Yamaha Steinberg USB ASIO) as the audio interface, you can select 44100 Hz.
  3. Choose a preset or select samples manually, then create the sequence data in XO.

2. Export WAV and MIDI from XO

  1. Open the Export WAV & MIDI panel.
  2. Click Render WAVs.
  3. Click Processed Kit. WAV files will be written to the export folder.
  4. Drag and drop Beat as MIDI to export the MIDI file.

3. Convert the MIDI file

  1. Open xo-to-seqtrak.
  2. Drag and drop the MIDI file exported from XO.
  3. Download the converted ZIP file.
  4. Extract the converted ZIP file before importing it into SEQTRAK App.

4. Import WAV files into SEQTRAK App

  1. Open SEQTRAK App.
  2. Open PROJECT / SOUND MANAGER.
  3. Open the SOUNDS tab.
  4. Drag and drop all WAV files exported from XO.
  5. Change the category for each WAV file.

    You may choose any category, but the following categories place the samples at the beginning of each slot group on SEQTRAK.

    SEQTRAK Slot Recommended Category
    SLOT 1 KICK
    SLOT 2 SNARE
    SLOT 3 CLAP
    SLOT 4 CLOSED HIHAT
    SLOT 5 OPEN HIHAT
    SLOT 6 TOM
    SLOT 7 SHAKER / TAMBOURINE
  6. Click IMPORT to import the WAV files.

5. Import MIDI patterns into SEQTRAK App

  1. Return to HOME and open GUI EDITOR.
  2. Select the DRUMKIT track, then click PATTERN.
  3. Click the MIDI IMPORT button.
  4. Select the MIDI file you want to import from the extracted ZIP files.
  5. Select the destination slot number.
  6. The pattern is now imported, and you can reproduce the same sequence data from XO on SEQTRAK.
